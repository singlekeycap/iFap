# iFap

An all-in-one app that has multiple sources of R-rated material.

### Features:
- OnlyFans + URL Scheme (onlyfans://<model username>)
- Fansly + URL Scheme (fansly://<model username>)
- Orange and black YouTube + URL Scheme (pornhub://<video id>)
- And more!
